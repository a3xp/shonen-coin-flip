# Shonen Coin Flip

Un simulatore di lancio della moneta realizzato in HTML, CSS e JavaScript, con animazioni ispirate agli anime shonen e ai manga giapponesi.

## Funzionalità

- Lancio casuale testa o croce
- Contatore separato per testa e croce
- Animazione 3D della moneta
- Effetti manga e shonen
- Particelle, flash, scie cinetiche e vibrazione dello schermo
- Effetto sonoro incorporato direttamente nella pagina
- Supporto per pulsante e barra spaziatrice
- Layout responsive per computer e smartphone
- Pulsante per azzerare i risultati

## Tecnologie

- HTML5
- CSS3
- JavaScript
- Web Audio e audio HTML incorporato

## Avvio locale

Non sono richieste installazioni.

1. Scarica o clona il repository.
2. Apri `index.html` con un browser moderno.
3. Premi **Scatena il lancio** oppure la barra spaziatrice.
